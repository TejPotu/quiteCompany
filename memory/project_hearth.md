Hearth is a tablet UI prototype (React + Vite) designed for people living with dementia. It has four screens: Home, Watch (TV remote), People (face recognition), Reminders.

**Why:** the user has stated explicitly that the design must be dementia-patient friendly, not generic-human friendly. This is a recurring framing.

**How to apply:** when designing/editing any screen, optimize for: very large text and imagery, minimal choice, one clear focal point, no clever multi-purpose components, plain language, obvious "what is happening right now." Strip anything that adds cognitive load even if it would be useful to a non-impaired user (extra rows, marketing copy, secondary metadata, decorative chips).
