---
name: Dementia-first UI bar
description: When iterating on Hearth UI, default to bigger/fewer/clearer over feature-rich
type: feedback
---

Default to bigger images, bigger names, fewer items, and an unmistakable "now playing / now happening" focal point. Cut secondary shelves, intent chips, and metadata that a typical user would appreciate but a dementia patient would find noisy.

**Why:** user reiterated that this is for a dementia patient, not a normal user, after seeing the initial Watch tab which had multiple shelves and small tiles. They want clean, only favourite shows, bigger images, bigger titles, and a clearly highlighted currently-playing show with episode info.

**How to apply:** for every screen, ask "would a dementia patient need this element to do the one thing this screen exists for?" If not, remove it. Prefer one hero focal point + one simple grid of large favourites over multiple shelves. Show episode/now-playing context prominently.
